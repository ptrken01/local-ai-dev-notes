## Setup Instructions: Using AI Prompts in Your Workflow

The core of this system is integrating prompts into your existing tools. You’ll want to use them with a tool like ChatGPT or Claude, and store the prompt outputs in a shared folder for client files or social media drafts.

### Step 1: Create a Prompt Library
Organize your 50 prompts into categories:
- Client Program Writing (e.g., "Write a 4-week beginner strength program")
- Social Media Content (e.g., "Create a post about the importance of rest days")
- Direct Messaging (e.g., "Write a DM to a client who hasn't responded in 3 weeks")

### Step 2: Use This Template for Consistency
Each prompt should start with:

```
You are an expert fitness coach. Your goal is to create content that converts.
Target audience: [describe your niche]
Format: [e.g., 4-week program, social post, email]

[Specific task here]
```

### Step 3: Automate Output
Use a simple script or workflow tool like Zapier or Make to auto-save outputs into a folder. For example:
```bash
# Example bash automation (optional)
mkdir -p ~/fitness-prompts/clients
echo "$prompt_output" > ~/fitness-prompts/clients/client_01_program.md
```

This setup lets you create 2–3 programs per hour instead of 1–2 per week, increasing your throughput by over 50%.
