## The Prompt Workflow

Here's a concrete example of how to use prompts in your workflow:

```python
def generate_workout_caption(exercise_type, client_progress, tone="motivational"):
    prompt_template = f"""
    Create a workout caption for {exercise_type} that focuses on {client_progress}.
    Tone should be {tone}. Include 1-2 emojis and end with a call-to-action.
    Keep under 150 characters.
    """
    return prompt_template

# Usage
caption_prompt = generate_workout_caption("deadlifts", "strength gains", "encouraging")
print(caption_prompt)
```

This produces a reusable template that generates consistent, personalized captions. Each prompt can be customized with specific variables like exercise type, client progress, or desired tone.
